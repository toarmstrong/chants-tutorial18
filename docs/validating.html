Validation

In Practice

Running the commands

  1. Open the terminal.
  2. Enter sbt console to launch the scala console in the terminal.
  3. Enter :load scripts/validate.sc to load the validation script.
  4. Enter validate("PageURN") replacing PageURN with the appropriate page urn (found in the last column of the dse).

To quit the scala terminal, use the command :q. To quit the terminal, use exit.

Checking the results

Once generated, validations live in chant/validation.

Check summary.md first. Are there errors in the dse?

Next, check dse-validation.md. Are the records consistent? Are there any errors?

Finally, check dse-verification.md. You may want to turn on markdown preview in Atom - Packages > Markdown Preview > Toggle Preview. Follow the link all dse relations of page.... Are all areas of the page covered? Are all the boxes fully formed?
